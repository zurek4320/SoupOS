# SoupOS
A simple retro hobby operating system

## How to run it?
You can use GRUB to change it into .iso file and then use Rufus to run it but it's not too useful OS anyways so I think it's better to copy all needed files (boot.asm, kernel.cpp, kernel_entry.asm, test.sh, zeroes.asm) and run test.sh on Linux to simulate SoupOS on QEMU.
